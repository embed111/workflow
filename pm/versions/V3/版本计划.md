## 1. 版本定位
- `V3` 聚焦小伙伴的专业边界、学习节奏和知识清理机制。
- 本版本不是简单增加角色数量，而是让现有角色更像真正专业岗位。
- 本版本也负责把测试资产责任边界与覆盖维护机制制度化，不再只靠 PM 临时盘点。
- `V3` 已于 `2026-04-15T16:21:04+08:00` 接替 `V2` 成为当前 active 版本，并在 `2026-04-17T11:35:00+08:00` 满足退出门槛后切交给 `V4`。

## 2. 进入前提
- `V2` 已把 PM 高频治理动作收轻
- 角色工作区基础可用性已在 `V1` 收稳

## 3. 版本目标
- 把角色专业边界、每日学习提示、知识清理和协作节奏做成稳定机制。
- 让每个小伙伴都有更明确的专业责任和成长路径。
- 让 `workflow_testmate / workflow_qualitymate / workflow_bugmate` 对测试、覆盖盘点和质量回归形成稳定分工。
- 把专家成长治理文件化，形成 `学习方法文件 + 能力总表 + 问题-方法映射 + 理论库建设清单` 的稳定入口，为后续“自我修炼功能”预留清晰契约。
