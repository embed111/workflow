- 让接口目录能力进入项目运营页、角色合同和工作台，不再只是底座治理页里的“可看不可用”能力。

- 为第二项目和 helper 的日常工作补一组最小消费 smoke 与 go/no-go，避免 `V7` 再次停在主题标题层。
- 把当前必须排期的界面扁平化、项目动态增删和版本前置排期补进 active 版本，不能再让这些硬要求游离在对话外。
