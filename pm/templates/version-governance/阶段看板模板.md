# 阶段看板模板

- version: `VX`
- updated_at: `YYYY-MM-DDTHH:MM:SS+08:00`

## 1. 阶段定义
| 阶段 | 含义 | 下一门示例 |
| --- | --- | --- |
| `draft` | 需求已入版本，但还没拆到可澄清粒度 | 需求澄清 |
| `clarified` | 范围和边界已收敛 | 方案设计评审 |
| `design_review_passed` | 方案通过评审 | 测试设计评审 |
| `test_review_passed` | 测试设计通过评审 | 开发 |
| `developing` | 开发实现中 | 功能演示 |
| `demo_passed` | 演示通过 | 测试/回归 |
| `testing` | 测试、回归与发布前验证中 | 发布收口 |
| `released_done` | 需求完成 | - |
| `blocked` | 当前阶段受阻 | 恢复动作 |
